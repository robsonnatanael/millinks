# Contribuindo

Contribuições são bem-vindas! Siga as etapas abaixo para contribuir com o **MilLinks**.

## Como Contribuir

1. **Faça um Fork do repositório** no GitHub.
2. **Clone seu fork** e crie uma nova branch de funcionalidade:

   ```bash
   git checkout -b feature/minha-funcionalidade
   ```

3. **Confirme suas mudanças**: Siga o padrão de **Conventional Commits** (ex: `feat: página de login`, `fix: alinhamento do cabeçalho`).

4. **Envie para seu fork**:

   ```bash
   git push origin feature/minha-funcionalidade
   ```

5. **Abra um Pull Request** para a branch `main`.

## Documentação e Internacionalização (i18n)

Ao contribuir com a documentação, certifique-se de fornecer as versões em **Inglês** (padrão) e **Português (BR)** do seu conteúdo.

Para um guia detalhado sobre como realizar traduções, consulte o nosso **[Guia de Localização](./localization.md)**.
