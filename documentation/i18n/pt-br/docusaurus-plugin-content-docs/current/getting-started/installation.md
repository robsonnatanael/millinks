# Instalação

Antes de iniciar, certifique-se de possuir os seguintes pré-requisitos:

## Pré-requisitos

- Node.js 24+
- Yarn (Recomendado) ou npm
- Git

## Clonando o repositório

```bash
git clone https://github.com/robsonnatanael/millinks.git
cd millinks

# Instalando dependências (Yarn é recomendado)
yarn install
# ou
npm install
```
