# Contributing

Contributions are welcome! Please follow the steps below to contribute to **MilLinks**.

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Clone your fork** and create a new feature branch:

   ```bash
   git checkout -b feature/my-feature
   ```

3. **Commit your changes**: Follow the **Conventional Commits** standard (e.g., `feat: login page`, `fix: header alignment`).

4. **Push to your fork**:

   ```bash
   git push origin feature/my-feature
   ```

5. **Open a Pull Request** to the `develop` branch.

## Documentation and Internationalization (i18n)

When contributing to the documentation, ensure that you provide both **English** (default) and **Portuguese (BR)** versions of your content.

For a detailed guide on how to perform translations, please refer to our **[Localization Guide](./localization.md)**.
